## Description
-   Console numeric matrix processor;
-   Functions:
    1. Add matrices;
    2. Multiply matrix to a constant;
    3. Multiply matrices;
    4. Transpose matrix;
    5. Calculate a determinant of matrix;
    6. Inverse matrix.
